日料好吃的秘密，一半在食材新鲜，另一半在**调味料**。但走进超市进口区一看——酱油分浓口淡口、味醂有本味醂和味醂风、醋有寿司醋和米醋……头都大了。

本文帮你一站理清日料厨房最常用的**十样调味料**，按"要不要买"分了三个梯队。

---

## 第一梯队·先买这几瓶

这几瓶是日料的骨架——买齐了就能做九成以上的家庭日料。

### 浓口酱油（しょうゆ）

日料里说的"酱油"默认就是指浓口酱油。

| 项目 | 说明 |
|------|------|
| 味道 | 咸鲜，有发酵的酱香，颜色深 |
| 用途 | 照烧酱、煮物、丼物、蘸料、炒饭……什么都用 |
| 推荐品牌 | キッコーマン（Kikkoman）特选丸大豆、ヤマサ |
| 替代方案 | 国内普通酿造酱油即可（黄豆酱油，挑配料表只有水+大豆+小麦+盐的） |
| 价格 | 约15~25元/瓶（500ml） |

> 不要买"配制酱油"——配料表里有水解植物蛋白液的那种，味道单薄。

### 味醂（みりん）

味醂是带甜味的料理酒，作用是给菜**增加光泽和甜味**，同时去腥。

| 项目 | 说明 |
|------|------|
| 味道 | 甜中带一丝酒味，不浓 |
| 用途 | 照烧酱、亲子丼、寿喜烧、玉子烧、煮物 |
| 推荐品牌 | 本味醂：九重味醂、マンジョウ（味の素） |
| 替代方案 | 糖半汤匙+清酒1汤匙（应急，效果打折扣） |
| 价格 | 约20~40元/瓶，能用很久 |

> **注意**：超市常见"味醂风调味料"（みりん風調味料），便宜但酒精度低、加了糖浆，和本味醂不是一个东西。但入门阶段味醂风也能用，不用纠结。

### 清酒（日本酒）

做菜用清酒不是让你喝的——便宜的那种"料理酒"就够。

| 项目 | 说明 |
|------|------|
| 味道 | 淡淡的酒香，去腥提鲜 |
| 用途 | 除腥气、让肉类更嫩、煮饭时加一点增香 |
| 推荐品牌 | 白鶴料理酒、月桂冠料理酒 |
| 替代方案 | 中国料酒也可以（但味道稍冲） |
| 价格 | 约15~30元/瓶 |

> 中国料酒和日本清酒在去腥上的效果差不多。如果家里常做中餐，料酒也够了。

## 第二梯队·有了更好

### 白だし（しろだし）

液体综合出汁——昆布和木鱼花的提取液+酱油+味醂+盐调好了，**冲水就是汤底**。

| 项目 | 说明 |
|------|------|
| 味道 | 鲜味为主，带淡淡酱油色 |
| 用途 | 乌冬、味噌汤的底、玉子烧、茶碗蒸、关东煮 |
| 推荐品牌 | ヤマサ昆布つゆ、キッコーマンうちのごはん |
| 替代方案 | 没有替代品，但有了它能省掉大多数出汁步骤 |
| 价格 | 约25~40元/瓶，做汤能用20次以上 |

白だし是"日料偷懒王"——有了它，乌冬面、味噌汤、亲子丼的汤底全是分分钟的事。[出汁的详细做法](../在家做出汁/)也可以看看。

### 淡口酱油（うすくちしょうゆ）

| 项目 | 说明 |
|------|------|
| 味道 | 比浓口酱油咸，但颜色淡。不是"淡味"，是"浅色" |
| 用途 | 做清透颜色的煮物、茶碗蒸、乌冬清汤 |
| 推荐品牌 | ヒガシマル |
| 替代方案 | 浓口酱油少放一点，颜色没那么讲究就行 |
| 价格 | 约20~35元/瓶 |

> **常见误区**：淡口酱油不是低盐酱油，它的盐分比浓口还高。只是颜色浅，让菜卖相更清亮。

### 米醋（米酢）

日本醋比中国醋酸度低、更柔和。

| 项目 | 说明 |
|------|------|
| 味道 | 柔和酸味，微甜 |
| 用途 | 寿司饭调味、拌菜、泡萝卜 |
| 推荐品牌 | ミツカン（Mizkan）米酢 |
| 替代方案 | 中国米醋也行，酸度差不多；做寿司饭的话加一点糖调成寿司醋 |

### 料理酒

其实就是做菜专用清酒，但可能加了盐（不能喝的那种）。

| 项目 | 说明 |
|------|------|
| 味道 | 和清酒一样但有点咸 |
| 用途 | 和清酒一样：去腥、软化肉质 |
| 替代方案 | 直接用普通清酒，或者中国料酒 |

> 如果你已经买了清酒做菜，料理酒可以跳过。它唯一的优势是便宜。

## 第三梯队·进阶了再考虑

### 味噌（みそ）

| 种类 | 特点 | 适合 |
|------|------|------|
| **信州味噌** | 咸甜适中，颜色淡黄，最百搭 | 日常味噌汤、腌渍 |
| **西京味噌**（白味噌） | 偏甜，颜色白，口感细 | 西京渍（味噌腌鱼）、甜品 |
| **赤味噌**（八丁味噌） | 咸重，颜色深红，发酵久 | 重口味味噌汤、炖菜 |
| **即溶味噌粒** | 冲泡即饮 | 出差、办公室应急 |

> 入门只买一罐的话选**信州味噌**。Hikari的即溶味噌粒也很方便，热水一冲就是味噌汤。

### 昆布（こんぶ）

| 项目 | 说明 |
|------|------|
| 产地 | 日高昆布（最好入门）、利尻昆布、罗臼昆布 |
| 用途 | 煮出汁、做昆布渍、放入味噌汤提鲜 |
| 怎么买 | 超市进口区干货，一袋能用很久 |
| 入门建议 | 日高昆布性价比最高，先买这个 |

### 木鱼花（かつおぶし）

| 项目 | 说明 |
|------|------|
| 形态 | 薄片状，遇热漂动 |
| 用途 | 做出汁、撒章鱼烧/豆腐上 |
| 怎么买 | 超市袋装削好的木鱼花（厚削更香） |
| 入门建议 | 买一袋袋装木鱼花就够了，先别碰整根鲣节（需要专门刨子） |

### 芝麻油/胡麻油

日料用芝麻油和中国的差不多，但更常用作**点缀**而非炒菜油：
- 煮好的味噌汤里滴几滴
- 和菠菜/豆芽凉拌
- 白芝麻直接撒或磨碎撒

## 入门采购建议

**如果只买3样**：浓口酱油 + 味醂 + 白だし

这三样能做的菜：乌冬面、味噌汤、亲子丼、照烧鸡肉、玉子烧、寿喜烧、猪排饭的汤汁

**如果想凑齐家庭日料套装**：上面3样 + 清酒 + 信州味噌 + 寿司醋（或米醋）

六样可以在家做九成以上的经典日料，厨房架子也不至于太挤。
