## 速冻猪排 + 一瓶酱

做猪排饭最麻烦的一步——裹粉油炸——已经被超市替你做了。**速冻裹粉猪排**用空气炸锅一热，剩下的就是组装。

两种吃法，看你今天想吃什么心情。

---

## 吃法一·炸猪排定食（とんかつ）

最简单，猪排炸好切条，配卷心菜丝和猪排酱。

### 食材

| 食材 | 用量 | 备注 |
|------|------|------|
| 速冻裹粉猪排 | 2块 | 超市冷柜，空气炸锅加热 |
| 卷心菜 | 1/4棵 | 切极细丝，冰水泡发沥干 |
| **猪排酱** | 适量 | ブルドックとんかつソース |
| 米饭 | 2碗 | — |
| 番茄/柠檬 | 可选 | 装饰和解腻 |

### 做法

1. **热猪排**：空气炸锅按包装说明加热（通常180°C 10~12分钟）
2. **切丝**：卷心菜切极细丝，冰水泡5分钟更脆
3. **切排**：猪排切条（2cm宽）
4. **装盘**：铺卷心菜丝→摆猪排→淋猪排酱

> 卷心菜丝秘诀：泡了冰水它会卷起来，蓬松好看。吃之前沥干就行。

## 吃法二·猪排盖饭（カツ丼）

猪排用甜咸汤汁和鸡蛋焖一下，盖在饭上。

### 食材（2碗）

| 食材 | 用量 |
|------|------|
| 速冻裹粉猪排 | 2块 |
| 鸡蛋 | 3个 |
| 洋葱 | 半个 |
| **白だし** | 2汤匙 |
| 水 | 200ml |
| 酱油 | 1汤匙 |
| 味醂 | 1汤匙 |
| 米饭 | 2碗 |

### 做法

1. **热猪排**：空气炸锅加热后切成条
2. **调汁**：白だし+水+酱油+味醂搅匀，倒入小锅
3. **煮汤汁**：加洋葱丝煮开
4. **放猪排**：猪排条平铺在汤汁里
5. **淋蛋**：鸡蛋打散但不要搅太匀，先淋一半
6. **焖**：盖盖小火30秒
7. **再淋蛋**：淋剩下的一半，再焖20秒
8. **盖饭**：连汁带蛋滑到热米饭上

> 蛋不要全熟就关火，半熟滑嫩的口感才对。
