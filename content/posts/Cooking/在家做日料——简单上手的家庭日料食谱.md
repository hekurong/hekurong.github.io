## 前言

日料给人的印象往往是"精致、麻烦、食材难买"。但事实上，很多经典日料在家就能做，而且不需要专业设备。本文整理了 **9 道经典家庭日料**，每道菜都附有：

- **完整食材清单**
- **详细步骤说明**
- **半成品 / 预制 / 酱汁替代方案**（超市和网购都能买到）

> 核心理念：**用预制酱汁和半成品降低门槛，用新鲜食材提升上限。** 两种路线都能做出好吃的日料。

## 一、日式咖喱饭（カレーライス）

最入门的日料，零失败率。

### 食材清单（2人份）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 洋葱 | 1个（中等） | 可用冷冻切好的洋葱丁 |
| 胡萝卜 | 1根 | 冷冻蔬菜混合包 |
| 土豆 | 1个 | 冷冻土豆块 |
| 猪肉/鸡肉 | 200g | 无需替代，超市现成切好 |
| **咖喱块** | 半盒（约45g） | **好侍百梦多咖喱**、**S&B金牌咖喱** |
| 水 | 400ml | — |
| 米饭 | 2碗 | 电饭煲现煮 |

### 做法

1. **切菜**：洋葱切丝，胡萝卜和土豆切滚刀块（约2cm），肉切一口大小
2. **炒洋葱**：锅中放油，中火炒洋葱至透明微焦（约5分钟），这是咖喱甜味的关键
3. **炒肉**：加入肉片翻炒至变色
4. **加蔬菜**：放入胡萝卜和土豆，翻炒1分钟
5. **加水炖煮**：倒入400ml水，煮沸后转小火，盖盖炖煮15分钟至蔬菜软烂
6. **关火加咖喱块**：关火，掰入咖喱块搅拌至完全融化
7. **再开火**：开小火煮3-5分钟至咖喱浓稠即可

### 小贴士

- **咖喱块推荐**：好侍百梦多（微甜、适合入门）、S&B金牌（经典辣味）、GAICO中辣（进阶）
- 加一小块**黑巧克力**或一勺**蜂蜜**，风味立刻提升一个档次
- 隔夜咖喱更好吃（"日之出咖喱"的说法）

## 二、寿司饭团（おにぎり）

最简单的日料，10分钟搞定早餐。

### 食材清单（4个饭团）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 热米饭 | 2碗 | 电饭煲现煮，用刚出锅的 |
| **寿司醋** | 2汤匙 | **自调**：醋2勺+糖1勺+盐半勺搅匀 |
| 海苔 | 4片 | 超市有售，也有即食海苔片 |
| 馅料（任选） | 适量 | 见下方推荐 |

### 推荐馅料

| 馅料 | 做法 | 难度 |
|------|------|------|
| 三文鱼 | 超市三文鱼片用盐煎熟，捣碎 | ★ |
| 梅子 | **梅干**直接用（超市有售） | ★ |
| 金枪鱼蛋黄酱 | 金枪鱼罐头+蛋黄酱拌匀 | ★ |
| 明太子 | 买的**即食明太子**直接用 | ★ |
| 鲑鱼子 | **盐渍鲑鱼子**（网购） | ★ |

### 做法

1. **调醋饭**：热米饭盛入大碗，淋入寿司醋，用饭勺切拌（不要搅拌，要"切"）
2. **准备手水**：一碗凉水+一点醋（防止米饭粘手）
3. **握饭团**：手沾醋水，取约100g米饭，中间压个小坑放入馅料，双手合拢轻握成三角形
4. **包海苔**：用海苔片包裹底部

### 小贴士

- **饭团的关键是"趁热握"**，冷饭握不紧
- 握饭团不要用力压，轻轻合拢即可，保持内部蓬松
- 可以用**保鲜膜**辅助塑形，更整齐

## 三、味噌汤（味噌汁）

日料的灵魂配角，每顿饭都要有。

### 食材清单（2碗）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| **味噌** | 1.5汤匙 | **Hikari即溶味噌**、**佐藤白味噌** |
| 昆布/海带 | 1小段（约10cm） | **即食昆布茶包**或**出汁包** |
| 木鱼花 | 一小把 | **木鱼花茶包**（便利店有售） |
| 嫩豆腐 | 半盒 | 可用绢豆腐 |
| 葱花 | 适量 | — |
| 装配料（任选） | 适量 | 海带芽、裙带菜、蘑菇、油豆腐 |

### 做法

1. **做出汁（だし）**：500ml水+昆布段，小火煮至将沸（不要沸腾），取出昆布，放入木鱼花，关火浸泡30秒，过滤
2. **融化味噌**：取少量出汁到小碗，用味噌滤网或勺子将味噌完全化开（不要直接把味噌丢进锅里，会结块）
3. **加入配料**：豆腐切小块放入汤中，小火加热至微沸
4. **调味**：倒入味噌液，**关火**（味噌不能煮沸，否则香味全跑）
5. **撒葱花**：盛碗后撒葱花

### 小贴士

- **味噌绝对不能煮沸**，这是最重要的规矩。保持在80°C以下
- 推荐囤一罐**综合出汁（合わせだし）**液体或粉状，跳过做出汁步骤
- 味噌品牌推荐：**西尾八丁味噌**（浓郁）、**白味噌**（清甜）、**信州味噌**（百搭）

## 四、日式煎饺（餃子）

一锅煎饺，外脆里嫩，蘸料灵魂。

### 食材清单（约20个）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 饺子皮 | 20张（约300g） | **直接买现成饺子皮**（超市冷柜） |
| 猪肉馅 | 200g | 超市现成肉馅 |
| 卷心菜 | 1/4棵 | 可用韭菜替代 |
| 葱 | 2根 | — |
| 姜 | 1小块 | **姜泥**（管装，超市有售） |
| 蒜 | 2瓣 | **蒜泥**（管装） |
| 酱油 | 1汤匙 | — |
| 芝麻油 | 1茶匙 | — |
| 盐、胡椒 | 适量 | — |
| 煎饺用油 | 1汤匙 | 普通食用油即可 |
| 水（煎饺用） | 50ml | — |

### 蘸料

| 材料 | 比例 |
|------|------|
| 酱油 | 2份 |
| 米醋 | 1份 |
| 辣油（可选） | 少许 |

### 做法

1. **调馅**：肉馅中加入切碎的卷心菜末、葱花、姜泥、蒜泥、酱油、芝麻油、盐、胡椒，**顺一个方向搅拌上劲**
2. **包饺子**：取适量馅料放在饺子皮中央，对折，先捏紧中间，再向两边捏褶子（新手直接对折捏紧也行）
3. **煎饺子**：平底锅倒油，中火摆入饺子，煎至底部金黄（约2分钟）
4. **加水**：倒入50ml水，**立即盖锅盖**，转中小火焖煮5分钟
5. **收汁开盖**：水蒸发完后，再煎1分钟让底部更脆
6. **倒扣出锅**：盘子扣在锅上翻转，底部朝上，完美冰花

### 小贴士

- 冷冻饺子直接煎也行，加水量多加20ml，焖煮时间加1分钟
- 想要**冰花效果**：加水时换成面粉水（1茶匙面粉+100ml水）
- 日式煎饺和中式水饺最大的区别：**日式是煎出来的，不是煮的**

## 五、日式乌冬面（うどん）

冬天来一碗热腾腾的乌冬，暖到心里。

### 食材清单（2碗）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 乌冬面 | 2包 | **超市冷冻乌冬**或**即食乌冬** |
| 昆布出汁/木鱼花 | 做法同味噌汤 | **浓缩出汁液**（一瓶搞定） |
| 酱油 | 2汤匙 | — |
| 味醂（みりん） | 2汤匙 | **可用料酒+糖**替代（1:1） |
| 配菜（任选） | 适量 | 见下方 |

### 推荐配菜组合

| 组合名 | 配菜 |
|--------|------|
| **天妇罗乌冬** | 炸虾天妇罗 + 萝卜泥 |
| **釜玉乌冬** | 生鸡蛋 + 葱花 + 酱油（拌着吃） |
| **咖喱乌冬** | 日式咖喱块化开做汤底 |
| **肉乌冬** | 薄切牛肉 + 洋葱 + 酱油味醂煮 |
| **狐狸乌冬** | 油豆腐 + 葱花（最朴素） |

### 做法（基础汤乌冬）

1. **做汤底**：400ml出汁 + 酱油2勺 + 味醂2勺，煮沸
2. **煮面**：另起一锅水，煮乌冬面（超市冷冻的按包装说明，通常2-3分钟）
3. **组合**：面捞入碗中，浇入热汤底，摆上配菜

### 小贴士

- **味醂很重要**，它给汤底带来特有的日式甜香。实在没有就用料酒+糖
- 冷冻乌冬面比干乌冬面口感好很多，推荐**丸大食品**或**东亚食品**的冷冻乌冬
- 乌冬面的汤底和味噌汤是两回事——乌冬用的是**酱油出汁**，不是味噌

## 六、日式炸鸡块（唐揚げ）

日料居酒屋的必点菜品。

### 食材清单（2人份）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 鸡腿肉 | 300g | 超市去骨鸡腿肉 |
| 酱油 | 2汤匙 | — |
| 味醂 | 1汤匙 | 料酒+糖 |
| 姜泥 | 1茶匙 | 管装姜泥 |
| 蒜泥 | 1茶匙 | 管装蒜泥 |
| 米酒/清酒 | 1汤匙 | 料酒 |
| 玉米粉/土豆淀粉 | 3汤匙 | **片栗粉**（日式土豆淀粉）更佳 |
| 油（炸用） | 适量 | 植物油 |
| 柠檬 | 1个 | — |
| 盐 | 少许 | — |
| **日式蛋黄酱** | 蘸食用 | **丘比蛋黄酱**（Kewpie） |

### 做法

1. **腌制**：鸡腿肉切3cm块，加入酱油、味醂、姜泥、蒜泥、米酒，抓匀腌制**至少30分钟**（冷藏过夜更入味）
2. **裹粉**：腌好的鸡肉均匀裹上玉米粉/片栗粉，抖掉多余粉
3. **炸制**：油温170°C（筷子入油有小气泡冒出），逐块放入，炸3-4分钟至金黄
4. **复炸**：捞出后油温升到180°C，再炸1分钟逼出多余油脂，外壳更脆
5. **装盘**：撒少许盐，挤柠檬汁，配蛋黄酱

### 小贴士

- **片栗粉**（日式土豆淀粉）比玉米淀粉炸出来更酥脆，推荐用
- 腌制时间越长越入味，但酱油盐度高的话别超过2小时
- **丘比蛋黄酱**是灵魂蘸料，日料的蛋黄酱和西式的完全不同（用蛋黄不用全蛋，微酸）

## 七、茶碗蒸し（日式蒸蛋羹）

滑嫩如布丁的日式蒸蛋。

### 食材清单（2碗）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 鸡蛋 | 2个 | — |
| 昆布出汁 | 300ml | 浓缩出汁液 + 水 |
| 酱油 | 1茶匙 | — |
| 味醂 | 1茶匙 | — |
| 盐 | 1小撮 | — |
| 配料（任选） | 适量 | 见下方 |

### 推荐配料

虾仁（去壳）、鸡肉末、银杏果、香菇片、鱼板（かまぼこ）、三叶草/葱花

### 做法

1. **调蛋液**：鸡蛋打散，不要打出太多泡沫（用筷子搅，不要用打蛋器）
2. **加出汁**：出汁+酱油+味醂+盐混合，**放凉**后倒入蛋液（热出汁会把蛋烫熟结块）
3. **过筛**：蛋液用细筛网过滤一遍，口感更细腻
4. **摆配料**：蒸碗底部放配料，倒入蛋液（碗的八分满）
5. **蒸制**：蒸锅水烧开后转**最小火**，碗口盖锡纸或盘子，蒸**12-15分钟**
6. **检验**：轻晃碗，蛋液微微颤动、不流动即可

### 小贴士

- **火候是关键**：大火蒸出来的蛋羹有蜂窝孔，小火慢蒸才滑嫩
- 出汁温度一定要凉，这是避免气泡的秘诀
- 超市有卖**茶碗蒸套装**（含碗+蛋液粉+配料），完全零难度

## 八、日式土豆沙拉（ポテトサラド）

百搭配菜，便当必备。

### 食材清单（3-4人份）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| 土豆 | 3个（中等） | — |
| 黄瓜 | 1根 | — |
| 洋葱 | 半个 | — |
| 火腿/培根 | 4片 | **即食火腿片** |
| 胡萝卜 | 半根 | 冷冻胡萝卜丝 |
| **丘比蛋黄酱** | 3-4汤匙 | 这个不能省 |
| 米醋 | 1汤匙 | — |
| 盐 | 适量 | — |
| 黑胡椒 | 少许 | — |

### 做法

1. **煮土豆**：土豆带皮煮熟（约20分钟），趁热去皮
2. **捣碎**：土豆用叉子压成粗泥（保留部分颗粒感，不要太细）
3. **处理蔬菜**：黄瓜切薄片撒盐腌5分钟挤干水分；洋葱切薄片泡冷水去辛辣
4. **混合**：土豆泥+黄瓜+洋葱+火腿丁+胡萝卜丝，加入蛋黄酱、醋、盐、黑胡椒拌匀
5. **冷藏**：冰箱冷藏30分钟更好吃

### 小贴士

- **丘比蛋黄酱**是这道沙拉的灵魂，不建议用其他蛋黄酱替代
- 土豆不要用料理机打，要手动压，保留颗粒感
- 日式土豆沙拉和西式最大的区别：**蛋黄酱量大**，而且加醋提味

## 九、寿喜烧（すき焼き）

冬日灵魂料理，甜咸汤汁涮牛肉，蘸生鸡蛋吃，一口入魂。

### 食材清单（2-3人份）

| 食材 | 用量 | 半成品替代 |
|------|------|-----------|
| **肥牛卷/牛肉片** | 300g | 超市火锅肥牛卷即可 |
| 老豆腐 | 1块 | 可用嫩豆腐，但老豆腐更吸汁 |
| 茼蒿/春菊 | 1把 | 超市有售，也可用菠菜替代 |
| 金针菇 | 1把 | — |
| 魔芋丝/蒟蒻丝 | 1袋 | 超市冷柜，也有结状的 |
| 洋葱 | 1个 | — |
| 大葱 | 1根 | — |
| 香菇 | 4-5朵 | — |
| **寿喜烧酱汁** | 见下方 | **万字寿喜烧酱**（现成瓶装） |
| **生鸡蛋** | 2-3个 | 溏心蛋也行，但生蛋蘸食是灵魂 |
| 乌冬面/米饭 | 适量 | 冷冻乌冬或白饭 |

### 寿喜烧酱汁（自制版）

| 材料 | 用量 |
|------|------|
| 酱油 | 4汤匙 |
| 味醂 | 3汤匙 |
| 糖 | 2汤匙 |
| 清酒/料酒 | 2汤匙 |
| 水 | 2汤匙 |

> **懒人方案**：直接买**万字（Kikkoman）寿喜烧酱汁**，超市进口区或网购都有，一瓶搞定。

### 做法

1. **调酱汁**：酱油+味醂+糖+清酒+水搅匀备用（或倒出一瓶盖现成酱汁）
2. **切菜**：豆腐切厚片，洋葱切圈，大葱斜切段，香菇顶部划十字花刀（美观），金针菇去根
3. **煎豆腐和葱**：厚底平底锅中火，不放油，干煎豆腐至两面微焦，大葱煎出焦色，盛出备用（这一步很关键，焦香味是寿喜烧的灵魂）
4. **煎肉**：锅中铺满牛肉片，**不要堆叠**，两面各煎5秒变色即可（不要煮老了）
5. **倒入酱汁**：将酱汁倒入锅中，煮沸
6. **下蔬菜**：放入茼蒿、金针菇、魔芋丝、香菇、煎好的豆腐和葱
7. **涮着吃**：肉变色后立刻夹出，蘸打散的生鸡蛋液吃
8. **煮乌冬收尾**：菜和肉吃完后，锅里剩的汤汁煮一把乌冬面，吸饱酱汁的乌冬是压轴

### 小贴士

- **生鸡蛋是灵魂**：蘸生蛋液能给滚烫的肉降温，同时增加滑嫩口感。用**可生食鸡蛋**（超市有标注"可生食"）
- **关西vs关东**做法不同：关西先煎肉再加酱汁（甘辛口），关东先煮汤底再涮肉（清淡口）。家庭做推荐**关西风格**，味道更浓
- 牛肉不要煮太久，**变色即捞**，老了就柴了
- 酱汁可以循环加——越煮越浓，吃完肉菜再煮面，一锅三吃

## 总结

日料在家做，核心就是三个字：**不将就**。

- 调料不将就——味醂、味噌、出汁这三样备齐，日料的"底味"就有了
- 食材可以将就——半成品和预制菜大大降低门槛，不用什么食材都从零开始
- 火候不将就——日料讲究的是"恰到好处"，多一分则过，少一分则欠

从咖喱饭和饭团开始，逐步挑战天妇罗和茶碗蒸，你会发现：日料其实没那么难。

**ご飯ですよ！（开饭啦！）**
